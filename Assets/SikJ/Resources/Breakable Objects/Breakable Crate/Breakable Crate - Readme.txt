Features

• Breakable crate prefab
• Scripted for customization
• Resizeable prefab
• 1024×1024px resolution UV mapped textures and bump
• Two sets of texture and normal maps (with and without moss)
• Layered textures, easy to change visual appearance
• Works well on Unity and Unity Pro
• Commented code
• Scenes and environment included (as shown in preview)
• Object is 1 mesh until broken, for performance
• Particle System on break

Easily drag and drop to the game scene. This prefab contains two models, a whole mesh and a fractured model of that mesh. When the objects fracture function is called it is replaced with the fractured model.
